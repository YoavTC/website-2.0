de_jaffa


de_jaffa is a CS:GO wingman map i made to improve my mapping skills. The map's main gamemode is for wingman, Counter Strike's version of 2v2. The map is based on Jaffa and Jerusalem in real life, and I think it looks really similar to them.

I used the Hammer Editor for source in order to create the map. The map uses already-existing textures and models made by Valve, but the map's layout and design were entirely done by me.
